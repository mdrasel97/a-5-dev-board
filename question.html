What is event bubbling in the DOM?
event bubbling, refers to the behavior where when an event occurs on an element, it triggers not only on that element but also propagates upwards through the DOM hierarchy, reaching its parent elements and ancestors until it reaches the topmost element, allowing event handlers attached to parent elements to also react to the event that originated on a child element.
